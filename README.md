# StayEase – Room Booking System

A full-stack room booking application where users can browse rooms, filter by price and location, and book rooms.

## Tech Stack

Frontend:
- React.js
- Axios
- CSS

Backend:
- Node.js
- Express.js
- JWT Authentication

Database:
- MySQL

## Features

- User Registration & Login
- JWT Authentication
- Browse hotel rooms
- Filter by price and location
- Book rooms
- View My Bookings
- Responsive UI

## Installation

### Backend

cd backend  
npm install  
npm start  

### Frontend

cd frontend  
npm install  
npm run dev  

## Database

Import the `room_booking.sql` file into MySQL.
